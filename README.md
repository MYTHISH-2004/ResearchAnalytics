# 🎓 Faculty Academic Analytics Platform

A web-based application that enables faculty to manage **students, attendance, marks, and academic insights** from a centralized dashboard.

The system supports data entry, automatic calculations, visualization, and report exports to help educators make informed decisions.

## 🚀 Features

- Secure login (email / username)
- Student management (add, search, filter, delete)
- Attendance entry with automatic percentage calculation
- Marks management (single & bulk entry)
- Dashboard with charts and summaries
- Identification of top performers
- At-risk student detection
- Department & subject analytics
- CSV export for records
- Responsive modern UI
